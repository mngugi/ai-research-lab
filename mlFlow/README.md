# MLflow Experiments 🚀

- This folder contains experiments using MLflow for tracking machine learning models.

## Structure

* scripts/ → Python scripts for experiments
* notebooks/ → Jupyter notebooks
* models/ → Saved models
* experiments/ → MLflow tracking data

## Run MLflow UI

`mlflow ui`

Then open:
http://127.0.0.1:5000

## Goal

Learn and apply experiment tracking for reproducible ML research.
